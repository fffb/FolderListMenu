# FolderListMenu  

## 功能：
在“打开”或“保存”对话框弹出时，按下 Ctrl+G，会在鼠标处弹出一个菜单，包含当前已打开文件夹的路径，点击菜单，会在“打开”或“保存”对话框中快速定位到对应路径。支持 Total Commander 和自定义常用文件夹。
